# Test Scripts

Manual test scripts for debugging and verification.

## Examples
- `test-openai.ts` - Test OpenAI API connection
- `test-supabase.ts` - Test Supabase connection
- `test-pdf.ts` - Test PDF generation
- `test-csv-parse.ts` - Test CSV parsing with sample file

## Usage
```bash
npx tsx scripts/tests/test-openai.ts
```

## Note
These are for manual testing, not automated test suites.
For automated tests, use Jest/Vitest in the `/src` folder.
