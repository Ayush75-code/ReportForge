# Database Seeds

Seed scripts for populating the database with test/demo data.

## Examples
- `seed-clients.ts` - Create sample clients
- `seed-reports.ts` - Create sample reports with analysis
- `seed-demo-agency.ts` - Create full demo agency setup

## Usage
```bash
npx tsx scripts/seeds/seed-clients.ts
```

## Note
Only run these in development/staging, never in production.
