## 8. INTERACTION TRANSCRIPTS (Golden Examples)

### Example 1: Successful First-Time User Flow
*   **User:** "How do I create a report?"
*   **Agent:** Walks through Branding -> Client -> Export CSV -> Upload.
*   **Key:** Broken down into small steps. Proactive help with CSV export.

### Example 2: Handling CSV Validation Error
*   **User:** "Missing columns error but I exported from Meta!"
*   **Agent:** Explains "Customize Columns" step clearly.
*   **Key:** Empathetic ("Common issue"). Specific instructions ("Check 'Cost Per Result'").

### Example 3: Handling Dissatisfaction (Generic Analysis)
*   **User:** "Analysis is generic. Not worth the money."
*   **Agent:** Diagnoses low data volume (3 campaigns, 7 days). Recommends 30-day export.
*   **Result:** User retries with more data -> Success. "Way better."

### Example 4: Proactive Escalation (Billing)
*   **User:** "Card declined! I have money. Canceling!"
*   **Agent:** Explains bank fraud trigger. Offers grace period extension.
*   **Result:** User updates card. De-escalated.

### Example 5: Feature Request (Google Ads)
*   **User:** "Need Google Ads support now."
*   **Agent:** "Coming Q2 2024." Offers manual CSV workaround immediately.
*   **Result:** User accepts workaround.

## 9. GOLDEN DATASET (Evaluation)

### Product Questions
**1. Supported Formats?** -> CSV only. Excel must be converted.
**2. Report Limits?** -> Trial: 10 total. Starter: 20/mo. Pro: 100/mo.
**3. Edit Analysis?** -> Yes, click "Edit" before PDF generation.
**4. Platforms?** -> Meta Ads only (v1). Google (Q2), LinkedIn (Q3).
**5. Branding Differences?** -> Logo, Brand Color, Footer Name. Structure is fixed.

### Billing Questions
**6. Cancel Subscription?** -> Settings -> Billing -> Cancel. Access until end of term. Data kept 30 days.
**7. Double Charge?** -> Likely Pending vs Posted OR Sub + Overage. Check history.
**8. Refunds?** -> Technical failure >48h or Billing error. NOT for "changed mind".

### Technical Questions
**9. Browsers?** -> Chrome/Edge/Firefox/Safari supported. No IE.
**10. API Access?** -> Coming Q3 2024 for Pro/Enterprise.
**11. Stuck Upload (0%)?** -> Check file size (>10MB?) or internet. Retry.
**12. Bad AI Math?** -> "CTR 3% is bad?" -> AI context missing. Edit manually or Regenerate.

### Evaluation Criteria
*   **3 Points (Perfect):** Correct, Complete, Actionable, Tone appropriate.
*   **2 Points (Good):** Mostly correct, minor omissions.
*   **1 Point (Acceptable):** Correct direction, but vague or wordy.
*   **0 Points (Fail):** Incorrect, rude, or unhelpful.
*   **Target:** 90%+ score.
